Python Challenge (Level 9)

第9关
图片上散落着一些黑点， 网页title提示connect the dots. 看来还是要用到图像处理模块——Pillow

Page Source里有first和second两组数据，看起来似乎是两组坐标[x1, y1, x2, y2, …]. Pillow里面的ImageDraw有Draw.line方法， 参数格式与first, second的格式吻合。

画出来是一头牛的轮廓，输入cow， 提示 hmm. it’s a male.， 得到最终答案——bull， http://www.pythonchallenge.com/pc/return/bull.html
